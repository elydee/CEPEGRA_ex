Insertion d'un string HTML

('<div class="bckgrd" >new paragraphe </div>')

  • Insertion à l'intérieur d'un élément
    • $('sélecteur').append('à insérer')
    • $('sélecteur').prepend('à insérer')
    • $('à insérer').appendTo('cible')
    • $('à insérer').prependTo('cible')
  • Insertion à l'extérieur d'un élément
    • $('sélecteur').before('à insérer')
    • $('sélecteur').after('à insérer')
    • $('à insérer').insertBefore('cible')
    • $('à insérer').insertAfter('cible')
  • Insertion autour d'un élément
    • $('sélecteur').wrap('à insérer')
    • $('sélecteur').wrapAll('à insérer')
    • $('sélecteur').wrapInner('à insérer')
  • Remplacement d'un élément
    • $('sélecteur').replaceWith('à insérer')

Insertion d'un élément HTML

var element = $("#intro")

  • Insertion à l'intérieur d'un élément
    • $('sélecteur').append(element)
    • $('sélecteur').prepend(element)
    • $(element).appendTo('cible')
    • $(element).prependTo('cible')
  • Insertion à l'extérieur d'un élément
    • $('sélecteur').before(element)
    • $('sélecteur').after(element)
    • $(element).insertBefore('cible')
    • $(element).insertAfter('cible')
  • Insertion autour d'un élément
    • $('sélecteur').wrap(element)
    • $('sélecteur').wrapAll(element)
    • $('sélecteur').wrapInner(element)
  • Remplacement d'un élément
    • $('sélecteur').replaceWith(element)
